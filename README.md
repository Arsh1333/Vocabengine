This app helps you build vocabulary the same way you naturally read. Instead of memorizing word lists, you paste real English content and interact with unfamiliar words directly. You can save words you want to revisit and revise later

## Instalation

Follow these steps to get the project running locally.

### Prerequisites

_Make sure you have the following installed:_
* [Node](https://nodejs.org/en) - Node.js (v18 or above recommended)
* [React](https://reactjs.org/) - JS Library.

npm or yarn

A modern web browser (Chrome / Firefox)]

1. Clone the repository:

```bash
git clone https://github.com/your-username/vocabengine.git
```
2. Navigate into the project folder:

```bash
cd vocabengine
```
4. Install dependencies:

```bash
npm install
```
5. Start the development server:

```bash
npm run dev
```
6. Open your browser and visit:

```bash
http://localhost:5173
```

## Usage

1. Read or paste English content in the app.

2. Save words you find unfamiliar.

3. Track: Total words collected

4. XP and level progression

5. Daily streaks

6. Use Revision Mode to review a limited set of random saved words.
